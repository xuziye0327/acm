数论
参考： [_随心所欲_][1]
题意很好理解， 就是从 ![$\[a, b\]$][2] 中取一个数 ![$x$][3]， 再从 ![$\[c, d\]$][4] 中取一个 ![$y$][5]， 问有多少种取法可以让 ![$(x + y) \mod p = m$][6]。
设 ![$f(a, b)$][7] 是从 ![$\[0, a\]$][8] 中取一个数和 ![$\[0, b\]$][9] 中取一个数满足上述条件的个数。
